# K距离

# Description

石油公司建设了*N*个两两联通的存储仓库，为了节省费用，只建设了*N*-1条道路。为了更好地规划运输方案，石油公司想知道两两仓库间的距离小于等于*K*的点对有多少对。

# Input

多组数据。第一行两个整数*N*, *K*。接下来N-1行每行三个整数*a*，*b*，*c*，表示a到b有一条长度为c的路。数据以*N=K=*0为结尾。

# Output

输出只包含一个整数，表示*K*距离点对数。

# Sample Input

```
5 4
1 2 3
1 3 1
1 4 2
3 5 1
0 0
```

# Sample Output

```
8
```

# Hint

数据范围：对于100%的数据，*N*<=233。

对于100%的数据，预处理。
